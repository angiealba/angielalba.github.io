HTML STRUCTURE



A HTML document has a standard structure which is composed of:

  • <!DOCTYPE html>: This is the document type declaration. It declares a document as being an HTML document.
  • <html> : This is called HTML root element and used to wrap all the code.
  • <head> : This contains the information about the HTML document.
  • <body> : This contains everything you want to display on the Web Page.
  • An HTML document can be created using any text editor. Save the text file using .html.

    • Displayed below is the proper HTML Structure:


    	<!DOCTYPE html>
		
	<html>
  
	<head>
	
		<title> HTML Structure </title>
	
	</head>
  
	<body>
	
		<h1> This is Heading 1.</h1>
      
		<p> This is a paragraph.</p>
  
	</body>
  
	</html>
